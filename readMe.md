# 일정관리 웹사이트 <뭐하냐?>

### 너무나 힘든 일정관리, 제대로 못하면 재현이가 화낸다!
#### 개발자 - 홍준혁 
#### 디자인 - 홍준혁
#### 기획 - 홍준혁
#### 도움을 주신 분 - 임재현

## 기능들 
-   Todo추가 / 삭제기능
-   이스터에그 추가(todo입력창에 뭔가를 입력하면 엄청난게...!)
-   심플한 디자인
-   공백입력 방지기능
-   추후에 더 추가할 예정입니다.
